Shopping Angular Web Application

Shopping Angular - Technologies Used

  • Node.js Express to serve it up
  • Search with options, ng-show, ng-hide and ng-init
  • Angular Web App using Routes, Controllers and Filters (custom and currency)
  • Use of ng-repeat, including filter:searchName, filter:search and orderBy:price
  • Full CRUD (Create, Read, Update and Delete)
  • Form submission
  • Table HTML creation
  • Services in Angular
  • Firebase for deployment and database using AngularFire
  • git version control

Shopping Angular - Description

This app is a full CRUD app (Create, Read, Update (change quanitity), Delete). This app is a shopping cart for tea, which was developed using Angular as a front-end client side and Firebase for the server side. This app calculates the funtionality of a shopping cart for teas. This app has Restful routes for checkout.
