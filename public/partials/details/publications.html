Publication API - Description

This app was a final exam for the Galvanize program and an exercise in writing code (by hand) for an 8 hour time frame. The Angular front-end side was deployed with Firebase and the backend Node side was deployed with Heroku. The app requirements included Restful routes, some styling, and completing two complete web apps. The Node Express side is an API that is loaded through MongoDB and was tested using Postman. The Angular side is a complete front-end web app that is a full CRUD (Create, Read, Update and Delete) using $http requests getting and posting data to the Node Express/ MongoDB App.
