Tuneset Web Application

Tuneset Web Application - Technologies Used

  • Node.js Express API through MongoDB
  • Facebook and Google Oauth, passport and cookie parser
  • Monk, Handlebars and MongoDB
  • Reg Ex for filtering
  • SASS preprocessor
  • Unirest for handling API response data from an API call
  • Vanilla Javascript with the DOM
  • Form submission
  • Column formatting
  • Abcjs plugin for sheet music rendering and MIDI downloading
  • Heroku deployment
  • git version control

Tuneset - Description

TuneSet accesses the Session.org API to get Traditional Irish abc music files, which include the name of the tune, the type of which it is a jig or a reel, the key, and other info about the tune. Tuneset takes this abc notatation and uses it with the abcjs plugin to render the sheet music for the tune and have the capability of downloading the MIDI for that tune. Tuneset accesses the API through the name of the tune on the first page or by key choices on the keys page. The key page includes querying the API for key, time signature, and page that the tune exists for the Session.org API. The API and returns a list of tunes where the user can choose a tune to add to a collection of tunes. This collection of tunes is added to the MongoDB.
